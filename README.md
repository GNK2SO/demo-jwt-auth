# Token-Based Authentication Demo Project

## Overview

I did this project to start my studies in TDD, I started with a simple example of token-based authentication.

### Features

 - Login
 - Refresh tokens
 - Save User
 - Get User

## How run?

run the command

```sh
./mvnw spring-boot:run
```

For the tests run the command:

```sh
./mvnw test

 ## Documentation
